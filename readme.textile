h1. survey

p. Survey is a Chrome extension that makes up a part of the Surfcave project, created in collaboratoin with Brad Troemel. This extension will send every image you see on the internet to "surfcave.com":http://surfcave.com. Surfcave debuted at "MAMA Showroom":http://www.showroommama.nl/projects/thesociallifeofthings.cfm in Rotterdam on November 4th, 2011.
